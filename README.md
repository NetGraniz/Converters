# Convert Lab

A lightweight collection of browser-based file converters and image utilities.

**Convert Lab** is an open-source project that provides fast, private, and easy-to-use conversion tools running entirely in your browser. No uploads, no server processing, and no tracking.

## Features

- Local file processing
- No server required
- GitHub Pages compatible
- Drag & Drop interface
- Batch processing
- ZIP archive download
- Responsive design
- Modern UI

## Privacy

All processing is performed locally in your browser.

Your files:

- never leave your computer;
- are never uploaded to any server;
- remain completely private.

## Available Tools

### WebP → PNG

Convert one or multiple WebP images into PNG format.

Features:

- Batch conversion
- Instant preview
- Individual download
- Download all files as ZIP
- Works offline

---

### Compress Photo

Reduce image size or resize images directly in your browser.

Supports:

- JPG
- PNG
- WebP
- Other browser-supported image formats

Compression modes:

- Target file size (KB)
- Target resolution
- Adjustable quality
- Output format selection

## Installation

Clone the repository.

```bash
git clone https://github.com/NetGraniz/Converters.git
```

Open the project.

```bash
cd Converters
```

Launch a local web server.

Example:

```bash
python -m http.server
```

or

```bash
npx serve
```

Open your browser:

```
http://localhost:8000
```
